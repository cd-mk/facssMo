핸드폰 연결

핸드폰 연결 상태 표시

핸드폰(스마트폰)으로 아래 링크에 접속하시거나 우측 QR코드를 촬영하여 페이지에 접속 후 인증 코드를 입력하세요. (안드로이드 : 크롬 브라우저, 아이폰 : 사파리 브라우저)

접속 링크 : https://nochibot.net/idcard
인증 코드 : A10-F48-C23-A01

접속 페이지 화면에서 [모니터링 연결] 버튼을 클릭하여 안내에 따라 예시와 같이 보이도록 핸드폰을 위치시켜 주시기 바랍니다.
핸드폰으로 촬영되는 영상은 응시생의 좌측 또는 우측에서 책상과 응시생의 측면이 보이도록 1~1.5미터 거리에 위치 시켜 주시기 바랍니다.

핸드폰이 연결되면 여기에 핸드폰 카메라 영상이 출력됩니다.

연결된 핸드폰 영상은 5초 후에 자동으로 종료되며, 시험장 입장 후 다시 시험감독관에게 전송이 시작됩니다.
